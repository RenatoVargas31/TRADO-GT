Este producto es una camiseta de manga larga de color azul, diseñada para ofrecer confort y estilo en cualquier ocasión. Fabricada con algodón de alta calidad, es suave al tacto y transpirable, ideal para el uso diario. Con un diseño minimalista, cuenta con un gráfico discreto en el pecho que le da un toque moderno y elegante.

La camiseta está disponible en todas las tallas, asegurando un ajuste adecuado para diferentes tipos de cuerpo. Está confeccionada con costuras reforzadas para mayor durabilidad y resistencia, incluso con el uso frecuente. Es perfecta para combinar con jeans, pantalones casuales o incluso bajo un blazer, lo que la convierte en una opción versátil para cualquier guardarropa.

Además, esta camiseta es fácil de cuidar, ya que es apta para lavar a máquina y mantiene su forma y color lavado tras lavado. Ideal tanto para hombres como para mujeres, es una prenda unisex que proporciona estilo y comodidad durante todo el año.
